ADVANTAGES:

	- Easy to work with 2D canvas.
	- Flexible game objects.
	- Multi-touch, State Management.
	- Simplicity.
	- Free and open source.
	- Extensive tutorials, and support forums.

DISADVANTAGES:

	- Does not have physics engine and collisions, then you should use Box2D or p2.js.
	- Not able to do 3D.
	- Not cloud enabled.
